## Install

```
$ pip install tocamelcase
```

## Why?

- Convert `non_camel_case` to `CamelCase`
- Clean and focused
- Actively maintained

## Usage

```py
import tocamelcase

print (tocamelcase.convert("non_camel_case"))
# Eg: non_camel_case → NonCamelCase
```
